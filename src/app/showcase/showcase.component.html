Standard: Automatic SVG Transformation

To make any visualization zoomable on mobile and desktop using EasyPZ, simply include:

<script src="https://code.easypz.io/easypz.latest.min.js"></script>

Note that all of this is possible without a single line of JavaScript code by the application developer. If you are wondering whether your visualizations would work well with the automated pan and zoom provided by EasyPZ, you can simply try it without any changes to your applications.
